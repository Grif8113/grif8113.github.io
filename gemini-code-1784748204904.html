LASER Project (Nine Gallon Nails)

Embedded Systems / Haptics

An assistive auditory alert system designed for hearing-impaired users. Features a central Arduino hub synchronized with a multi-motor tactile haptic pillow array to translate acoustic cues into real-time physical warnings.

Architecture Arduino, Central Hub Sync
Actuation Multi-motor Array, Polarity Switching
Display / UI Custom SPI/I2C OLED Driver
Failure Analysis & Engineering Iteration: Early prototypes experienced catastrophic overcurrent spikes that damaged microcontrollers due to unregulated motor stall currents. Redesigned to integrate active overcurrent protection, H-bridge drivers, and consolidated actuator channels on custom SMT PCB layouts.
